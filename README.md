# Atakan Reklam Dijital Katalog

Bu proje, statik dosyalarla çalışan modern bir dijital ürün kataloğu sitesidir.

## Özellikler
- Mobil uyumlu tasarım
- Ürün arama ve kategori filtresi
- Ürün detay ekranı
- WhatsApp üzerinden ürün seçimi
- Admin paneli ile ürün ve duyuru yönetimi

## Yerelde çalıştırma
Aşağıdaki komutla yerelde test edebilirsiniz:

```bash
python -m http.server 8000
```

Ardından tarayıcıda http://127.0.0.1:8000/ adresini açın.
