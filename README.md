# Atakan Reklam Dijital Katalog

Bu proje, statik dosyalarla çalışan modern bir dijital ürün kataloğu sitesidir.

## Özellikler
- Mobil uyumlu tasarım
- Ürün arama ve kategori filtresi
- Ürün detay ekranı
- WhatsApp üzerinden ürün seçimi
- GitHub Pages ile yayınlanabilir yapı

## GitHub Pages için
1. Bu klasörü GitHub üzerinde bir repo olarak yükleyin.
2. GitHub repo ayarlarında Pages bölümünden ana dalı seçin.
3. Site adresi şu yapıda olur:
   https://<kullanici-adiniz>.github.io/<repo-adiniz>/

## Yerelde çalıştırma
Aşağıdaki komutla yerelde test edebilirsiniz:

```bash
python -m http.server 8000
```

Ardından tarayıcıda http://127.0.0.1:8000/ adresini açın.
